## 简介

主题的使用仍然有一定的门槛，为了方便新手快速入门，尤其是非前端开发者，reco 主题尝试基于 [revili](https://revili.recoluan.com/) 开发可视化搭建平台。

```bash
# 全局安装 revili
pnpm add -g revili@next

# 安装 reco 主题套件
revili add revili-kit-vuepress-reco@next

# 启动可视化搭建平台
revili start
```
